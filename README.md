# springboot-user-management-jwt
A secure RESTful backend application built with Spring Boot implementing User CRUD operations, Spring Security authentication, DaoAuthenticationProvider, and JWT token-based authorization.

# App Description
-Implemented User Controllers to handle CRUD operations on User Entity.
-Developed a custom JWT layer in SecurityFilterChain.
-Coded a custom UserDetailsService.
-Used DaoAuthenticationProvider for Authentication process.
